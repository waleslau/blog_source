nvm 的全名是 Node Version Manager，是一个 node.js 的版本管理器，可以帮助用户通过命令行快速安装和使用不同版本的 node。nvm 可以在任何兼容 POSIX 的 shell（sh、dash、ksh、zsh、bash）上工作，尤其是在 unix、macOS 和 windows WSL。
